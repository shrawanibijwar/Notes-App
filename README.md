# 📝 Notes App

A simple Notes App built using Node.js, Express, and EJS.

## 🚀 Features
- Create notes
- Read notes
- Edit note title
- Delete notes (AJAX without reload)

## 🛠 Tech Stack
- Node.js
- Express.js
- EJS
- Tailwind CSS

## 📂 Project Structure
- views/ → UI templates
- public/ → static files
- files/ → stored notes

## ▶️ How to Run
1. Install dependencies
   npm install

2. Start server
   node index.js

3. Open in browser
   http://localhost:3000
